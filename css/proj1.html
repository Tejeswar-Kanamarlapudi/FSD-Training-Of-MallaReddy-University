Css day 1


HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It provides the basic building blocks for web pages, defining elements like headings, paragraphs, links, images, and more

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall visual appearance of web pages.
